# getAsinCode
get ASIN code from amazon.co.jp and mnrate.com

# Riquire 
Python3.6 and upper

# Requirements
os  
bs4  
urllib  
pandas  
xlwt  
xlrd

# How to run
1. Put inputUrl.xls in same folder with getASINfromUrl.py
2. Run code:  
`pip install --upgrade pip`
`pip install -r requirements.txt`  
`python getASINfromUrl.py`
